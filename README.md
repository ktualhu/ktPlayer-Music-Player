# ktPlayer - Music-Player
Music player on Java 8.

My test project on Java 8 with using JavaFx.

The following libraries were using:
  - mp3agic
  - jfoenix

How to run:
  - open cmd
  - come(cd) to target folder
  - run .jar file
  
How to use:
  - open folder with your music(left-top icon)
  - listening the music
